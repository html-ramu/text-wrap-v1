# Telugu Text Wrapper Pro

A specialized web tool designed to wrap Telugu text around images in various creative shapes.

## Features
* **Shapes:** Support for Circle, Square, 16-Gon, and Flower shapes.
* **Text Wrapping:** Automatically wraps Telugu text around the contour of the chosen shape.
* **Customization:** * Full RGB Color Picker for text and borders.
    * Image positioning (Zoom, Move Up/Down/Left/Right).
    * Adjustable text sizes.
* **Download:** Export your final design as a high-quality PNG.

## How to Use
1.  **Upload:** Click "Choose Image" to upload your photo.
2.  **Select Shape:** Choose a shape (e.g., Square, Circle) to apply the border.
3.  **Edit:** Paste your Telugu text in the box.
4.  **Customize:** Use the color pickers to change the border and text colors.
5.  **Download:** Click "Download" to save the image to your device.
